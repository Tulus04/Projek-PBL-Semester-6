🔴 Permasalahan

Absensi Manual

Tanda tangan kertas mudah dipalsukan, rekap lambat, data tidak real-time

🤝

Titip Absen

Tidak ada verifikasi identitas — mahasiswa bisa minta teman untuk absen

📍

Tidak Ada Lokasi

Tidak bisa memastikan mahasiswa benar-benar di kelas saat presensi

📊

Rekap Sulit

Dosen harus input manual, rentan salah, export data tidak tersedia

✅ Solusi: 3-Layer Verifikasi

🔲 Layer 1 — Scan QR Code
QR unik per sesi + OTP 6-digit
📍 Layer 2 — GPS Verification
Haversine formula + Anti Mock Location
🧑 Layer 3 — Face Recognition
MobileFaceNet 192-D + Server-side Cosine Similarity

Ketiga layer harus lolos untuk presensi diterima → Anti Titip Absen 100%
