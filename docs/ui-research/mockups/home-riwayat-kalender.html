Riwayat dalam Kalender

Format V3, isi diganti riwayat
Week strip = ringkasan minggu

7 hari, tiap hari diberi tint + dot sesuai status dominan (hadir hijau / izin kuning / alpa merah). Hari ini di-outline. Tap hari → agenda berubah.

Agenda = kelas di hari terpilih

Frame 1: Sabtu (2 kelas, hadir). Frame 2: Selasa (alpa). Menampilkan kelas + jam + pertemuan + status — kejadian yang SUDAH lewat.

Datanya SUDAH ADA ✓

AttendanceRecord punya scannedAt + status + courseName. Logika warna status dominan sudah ada di history_calendar_view.dart — tinggal reuse. Bisa dibangun sekarang, tanpa fitur baru.

Tap "Kalender penuh" → tab Riwayat

Preview minggu di beranda, detail bulan-penuh di Riwayat. Pola "preview di home, detail di tab".

Pertimbangan
Overlap dengan kalender Riwayat

Tab Riwayat sudah punya kalender bulan-penuh. Versi beranda = ringkasan minggu (glanceable). Overlap sehat, tapi sadari ada 2 kalender.

1 fetch ringkasan minggu

Beranda perlu ambil riwayat minggu ini (atau reuse data history yang sudah di-fetch). Sedikit nambah loading.
