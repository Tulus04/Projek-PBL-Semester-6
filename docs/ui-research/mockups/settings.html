Mode Verifikasi Wajah

Tentukan apakah face verification wajib untuk submit presensi atau opsional (mahasiswa bisa skip).

Opsional Recommended

Mahasiswa boleh submit tanpa face verify (tetap GPS + OTP). Cocok untuk masa transisi atau pengembangan. Setiap submit tetap tercatat di audit log apakah pakai face atau tidak.

Wajib (required) Strict

Setiap submit WAJIB lulus face verification (similarity ≥ threshold). Mahasiswa belum register wajah tidak bisa presensi sama sekali.

Threshold Cosine Similarity

Ambang batas similarity score untuk MobileFaceNet 192-d. Lebih tinggi = lebih ketat tapi false-reject naik.

Confidence threshold

Default: 0.65 (LFW benchmark MobileFaceNet). Min: 0.50 · Max: 0.95.
⚠ Jangan balik ke 0.75 — nilai lama untuk landmark heuristic embedding yang sudah ditinggalkan.

0.65
