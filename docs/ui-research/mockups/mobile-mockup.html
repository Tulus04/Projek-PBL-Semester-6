Cara Pakai Mockup Ini

Mockup ini adalah preview HTML statis menggunakan design tokens persis dari lib/core/theme/app_colors.dart + font Plus Jakarta Sans + Inter via Google Fonts + ikon Lucide. Tidak menggunakan TailwindCSS — semua raw CSS dengan CSS variables.

Yang bisa kamu evaluasi:

  • Composition & layout — penempatan card, hierarki, padding, spacing rhythm
  • Color usage — primary, success, warning, danger consistency
  • Typography — Plus Jakarta Sans untuk heading, Inter untuk body
  • Iconografi — Lucide outlined, ukuran konsisten
  • Micro-interaction — pulse, scale, slide-up (lihat hero session badge, success icon, consent modal)

Yang tidak bisa kamu evaluasi (limitasi HTML mockup):

  • Touch interaction (tap ripple, swipe gesture)
  • Camera live feed (mockup pakai gradient placeholder)
  • Native performance (animasi GPU-accelerated Flutter)

Setelah review, kasih feedback ke Cascade tentang:

  • Screen mana yang OK untuk implement ke Flutter?
  • Adjustment apa yang dibutuhkan (warna, spacing, copy text)?
  • Skip screen mana, atau tambah screen apa?
