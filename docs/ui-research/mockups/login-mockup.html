Apa yang Berubah di AFTER

Hapus trust badge sama sekaliTier 1
BEFORE: ada pill "Data terenkripsi · UU PDP compliant" di footer panel kiri.
AFTER: dihapus sama sekali — cukup tinggalkan copyright. Trust signal sudah tersirat lewat brand logo Politani + 3 fitur keamanan di atas. Pill jadi noise karena user yang sampai ke login sudah trust app-nya.
Bahasa deskripsi fitur lain di sebelah kiri tetap formal sesuai konteks akademik kampus.
Loading button — 3 alternatifTier 1
Tidak lagi ganti teks jadi "Memproses...". Klik 3 tombol Loading di toolbar:
A. Spinner kiri + teks "Masuk" tetap.
B. Spinner only (teks hilang).
C. Progress bar tipis di atas card, tombol utuh.
Page Enter AnimationTier 1
Panel kiri slide-in dari kiri (400ms), panel kanan delay 150ms. Konsisten dengan dashboard.
Stagger Feature ListTier 1
3 feature highlights muncul bertahap (280ms / 360ms / 440ms).
Email AutofocusTier 1
Field email otomatis fokus saat halaman load. Tidak perlu klik dulu sebelum ketik.
tabIndex Show/HideTier 2
Tombol mata password punya tabindex="-1" — keyboard user dari password langsung ke "Masuk".
CapsLock DetectorTier 2
Klik "CapsLock Aktif" — warning kuning muncul. Pesan: "CapsLock aktif — cek lagi password kamu sebelum lanjut."
Drift Blur CirclesTier 2
Blur kuning + putih di panel kiri pelan-pelan drift (18-22s loop). Sub-perceptual, nambah depth.
Placeholder EmailTier 2
BEFORE: contoh@email.com → AFTER: nama@politanisamarinda.ac.id (web cuma admin/dosen, jadi format kampus).
