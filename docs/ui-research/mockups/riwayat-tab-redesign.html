Audit Tab Riwayat

Sudah matang — fokus sambungan + poles
Sudah ada & bagus ✓

Hero 5-stat + progress, toggle Daftar/Kalender, filter chip, smart-date group, bottom sheet detail, heatmap kalender. Tidak perlu rombak.

Sambungan dari beranda

"Lihat semua" di beranda → tab Riwayat otomatis mode Kalender + bulan/tanggal terpilih sesuai preview. Continuity terjaga.

Kalender bulan penuh + detail tanggal

Tap tanggal → muncul daftar kelas hari itu di bawah kalender (atau bottom sheet). Multi-dot kalau >1 sesi dalam 1 hari.

Poles copy (rule 09)
Sub-judul appbar

"Catatan kehadiran semesterku" — sudah ramah & pakai "ku". Pertahankan.

Keputusan: detail tanggal di mana?

Kalender beranda → bottom sheet. Kalender Riwayat existing → juga bottom sheet. Di mockup ini saya tunjukkan inline di bawah kalender (frame kiri). Mana yang kamu pilih: inline atau bottom sheet?

Default mode tab Riwayat?

Saat dibuka via nav (bukan dari beranda) — default Daftar (existing) atau Kalender? Saat dari beranda → selalu Kalender.
