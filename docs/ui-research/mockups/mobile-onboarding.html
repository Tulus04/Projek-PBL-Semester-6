Cara Kerja Presensi

Tiga langkah verifikasi memastikan kehadiran kamu valid dan aman dari titip absen.

1. Scan QR Code Sesi

Dosen tampilkan QR di kelas, kamu scan via aplikasi untuk konfirmasi sesi yang benar.

2. Verifikasi Lokasi GPS

Pastikan kamu benar-benar di area kampus dengan radius geofence yang ditentukan dosen.

3. Face Recognition

Pastikan kamu sendiri yang absen, bukan orang lain pakai HP-mu.

Implementasi Notes
  • Trigger: cek SharedPreferences.hasSeenOnboarding di splash. Kalau false → /onboarding, kalau true → /login.
  • Skip behavior: tombol "Lewati" set flag dan navigate /login langsung.
  • Get Started behavior: tombol "Masuk Sekarang" set flag dan navigate /login.
  • Animation: PageView Flutter dengan transition slide horizontal 300ms ease-in-out.
  • Tidak ada consent UU PDP di sini — consent biometrik muncul di Face Register screen (existing rule 04-security-and-privacy.md Section B.5).
