Frame 2 · Percakapan Aktif User bubble · AI bubble (markdown) · typing dots · loading
Detail UX User bubble warna primary (kanan, radius bawah-kanan kecil). AI bubble putih + border (kiri, radius bawah-kiri kecil). Response AI pakai markdown ringan (bold, italic, ordered/unordered list) — parser custom tanpa library tambahan. Typing indicator 3 dot bounce staggered + teks "Menganalisis data...". Send button berubah jadi spinner saat menunggu response. Auto-scroll ke bawah setelah pesan baru.
