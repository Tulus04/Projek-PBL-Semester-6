Perbaikan #3 Hero icon: hilangkan gold glow + animasi float

Catatan: gold glow adalah signature visual brand Politani (lihat design system rule). Tapi kalau Anda merasa terlalu busy / mengganggu fokus, bisa di-disable. Plus tambah animasi float naik-turun 2.4s loop supaya icon terasa "alive" + bayangan tanah subtle untuk depth illusion.

📋 Yang Akan Saya Ubah Kalau Anda Approve

  1. Screen 1 hero icon background — di file lib/features/onboarding/screens/welcome_step.dart (atau nama serupa). Ganti gradient [primary, primaryDark] → solid primarySurface (#EAF3FF). Icon like_bold tetap primary.
  2. Screen 1 feature chips — tambah Row 3 pill (QR / GPS / Face ID) tepat di bawah deskripsi paragraph. Pakai SemanticIcon dengan variant action, warning, success.
  3. Screen 3 icon swap — di lib/features/onboarding/screens/get_started_step.dart. IconsaxPlusBulk.airplane → IconsaxPlusBulk.rocket.
  4. Hilangkan gold glow blob (opsional) — hapus Positioned Container yang punya RadialGradient(accentSoft → transparent) di hero icon container. ⚠️ Trade-off: brand signature Politani hilang dari hero. Pertimbangkan kalau benar-benar perlu.
  5. Animasi float naik-turun — bungkus HeroIconBox dengan AnimatedBuilder + AnimationController(duration: 2400ms) + Tween(begin: 0, end: -12).animate() looping. Tambah subtle drop shadow di bawah yang scale 1.0 ↔ 0.85 sinkron dengan position.

Total perubahan: ~30-40 baris di 2-3 file Flutter (jika semua diapply). Animasi pakai SingleTickerProviderStateMixin di onboarding screen. Estimasi 10-15 menit eksekusi + verify flutter analyze.
