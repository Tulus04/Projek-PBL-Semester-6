Detail Perubahan

Apa yang Berubah?

7 area perubahan visual — tanpa mengganti warna primary brand TRPL. Fokus pada depth, accent pendamping, dan micro-interaction.

01 — Accent Color

Tambah Amber #F59E0B

UI sekarang 1-tone biru. Tambah Amber sebagai accent kedua untuk highlight (notifikasi count, "menunggu review", focal interactive element). Tetap pertahankan primary biru TRPL untuk brand.

+ Brand + Accent

02 — Depth & Shadow

Layered Shadow Ala Linear

Saat ini card pakai shadow tunggal yang flat. Ganti ke layered shadow: 0 1px 2px rgba(0,0,0,0.04), 0 0 0 1px rgba(15,23,42,0.04). Hover state lift 2px + shadow membesar.

03 — Border Radius

16px → 20px

Card radius naik dari 16px ke 20px, icon container 11px. Lebih friendly modern, match Stripe Dashboard & Linear. Button tetap pill (999px) — sudah perfect.

04 — Hero Header

Gradient Welcome Card

Page header dashboard biasa → hero card gradient primary-dark to primary, dengan amber glow di pojok. Live indicator + summary highlight. Tombol & input tetap solid (rule design tidak dilanggar).

05 — Duotone Icon

Icon Box Berbintik Warna

KPI card icon polos di kanan atas → duotone icon box 38x38 dengan background warna soft (10% opacity). Visual hierarchy lebih jelas, KPI value lebih impactful.

06 — Trend Indicator

+12% / -3 Indicator Pill

KPI card cuma tampil angka saat ini. Tambah trend indicator pill di bawah (mis. ▲ +12% vs minggu lalu). User langsung tahu tren, bukan sekadar snapshot.

07 — Micro-interaction

Hover Lift + Cubic Easing

Card hover sebelumnya cuma shadow tumbuh. Tambah translateY(-2px) + smoother easing cubic-bezier(0.4, 0, 0.2, 1). Topbar pakai backdrop-blur ala Vercel. Nav active pakai indicator bar kiri.

Yang TIDAK Diubah

  • Primary color #5483AD — identitas brand TRPL Politani tetap.
  • Typography — Plus Jakarta Sans + Inter sudah optimal, tidak ganti.
  • Library — Tailwind, SweetAlert2, Lucide, Recharts tetap.
  • Tombol warna-warni mencolok — rule design tidak boleh, accent amber hanya untuk highlight pasif (badge, focal element), BUKAN tombol gradient.
  • Glassmorphism agresif — kecuali backdrop-blur halus di topbar (sudah ada di Vercel/Linear).
  • Dark mode — scope creep, ditunda untuk versi berikutnya.
